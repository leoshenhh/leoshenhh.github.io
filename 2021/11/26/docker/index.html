docker 命令

  • docker run

运行一个docker实例

  • docker ps

列出所有正在运行的docker实例

  • docker kill

关掉一个正在运行的docker容器

  • docker rm hello-world

删除一个docker镜像

docker 安装 mysql

  • --name 后面接 mysql名字（自定义）

  • -e 为设置环境变量

  • MYSQL_ROOT_PASSWORD 后面接 ROOT 用户的密码

  • -d 为开启守护进程

  • mysql: 后面接mysql 版本

docker 安装 postgres

  • 参数参考上面mysql
  • -p 后跟 主机端口:容器端口
